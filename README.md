## 📁 Projets

### 💪 HealthApp Web
Application web minimaliste pour le suivi quotidien de santé.

**Features:**
- 💧 **Eau** — Tracker les verres d'eau (objectif: 8/jour)
- 👟 **Pas** — Tracker les pas (objectif: 10 000/jour)
- ⚖️ **Poids** — Enregistrer et historique du poids

**Caractéristiques:**
- ✅ Responsive design (iPhone, mobile, desktop)
- ✅ Données sauvegardées en localStorage
- ✅ Reset automatique quotidien
- ✅ Export des données en .txt
- ✅ Interface minimaliste et rapide
- ✅ Zéro serveur, fonctionne hors ligne

**Accès:**
```bash
# En local
open healthapp/index.html

# Via GitHub Pages (quand déployé)
https://sacharm7.github.io/famille-carrier/healthapp/
```

**Données:**
Les données sont stockées localement dans le navigateur (localStorage). Aucun serveur requis.
